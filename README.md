# Strona dla baru sushu
Przykładowe wykonanie strony www dla fikcyjnego baru sushi. 
Strona jest przykładem wykonania serwisu dla restauracji, zawiera ona strone główną oraz dwie podstrony przeznaczone do opisu restauracji oraz do menu.
Wszelkie użyte na stronie dane są fikcyjne i służą wyłącznie prezentacji umiejętności.
